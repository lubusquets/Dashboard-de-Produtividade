# :chart_with_upwards_trend: Dashboard de Produtividade

Este projeto consiste no desenvolvimento de um dashboard interativo no Power BI para análise da performance de operadores, permitindo acompanhar indicadores de produtividade, qualidade e tempo de atendimento de forma visual e intuitiva.

# :dart: Objetivos
1. Monitorar a performance dos operadores.
2. Comparar indicadores de produtividade e qualidade.
3. Facilitar a análise por período e por operador.
4. Construir um dashboard profissional utilizando boas práticas de design.
